
#####################################
#####  Creating a new analysis  #####
#####################################

1) Go to folder the analysis will reside.
2) Clone base analysis scripts.
      git clone https://github.com/khegazy/UED_analysis.git
3) Create folders for holding results, run lists, etc.
      UED/preProcessing
        runLists
        results
      UED/staticDiffraction
        plots
      UED/mergeScans
        runLists
        results
      simulations
        xyzFiles
